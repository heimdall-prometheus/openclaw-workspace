## 4. "BEWOHNER"-WORT-TIC (Reduktion: 81× → ~40×)

### Ersetze "Bewohner" in folgenden Kontexten:

#### Allgemeine Formulierungen:
- „die Bewohner der Plattform" → „die Leute auf Nova Ventus"
- „alle Bewohner" → „alle hier" / „die Gemeinschaft"
- „neue Bewohner" → „Neuankömmlinge" / „neue Leute"
- „die Bewohner hatten Wasser" → „die Menschen hatten Wasser"

#### Kapitel 4, Zeile ~420:
**Original:** „Die Bewohner hatten Wasser, weil ein Ingenieur eine Dichtung gewechselt hatte."

**Ersetzen mit:** „Jeder dritte hier filmt mit seinem Handy."

#### Kapitel 7, Zeile ~220:

**Ersetzen mit:** „Die Leute werden nervös."

#### Kapitel 13, Zeile ~780:

**Ersetzen mit:** „Die Menschen lasen den Satz und schwiegen."
